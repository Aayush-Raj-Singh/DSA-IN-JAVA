## 🚀 How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/<Aayush-Raj-Singh>/DSA-IN-JAVA.git
   cd DSA-IN-JAVA
   ```
2. Open in your favorite IDE (e.g., IntelliJ or VS Code).
3. Explore by topic or problem.
4. Compile and run using:
   ```bash
   javac <filename>.java
   java <filename>
   ```
